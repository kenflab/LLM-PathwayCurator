# Paper artifacts (reproducible figures)

This folder contains minimal, reproducible scripts and fixed outputs used to generate paper figures.

## Fig2 (risk–coverage)
Inputs:
- report.jsonl (per cancer/method) + labels.tsv (R1_only)

Outputs:
- risk_coverage.tsv
- Fig2 plots (single cancer + multipanel)

See FIGURE_MAP.csv for the exact script↔input↔output mapping.
